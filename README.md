# Анализ поиска по картинкам в Яндекс.Картинках: desktop vs touch

Тестовое задание по продуктовой аналитике.

## Задача

Проверить гипотезу о том, что интересы пользователей, использующих поиск по
картинкам на мобильных устройствах и на компьютерах, заметно отличаются.

## Содержание репозитория

- `аналитика.ipynb` — полный анализ (Python/pandas, SQL через pandasql, визуализации)
- `data.tsv` — исходные данные (семпл запросов к Яндекс.Картинкам за 1–21 сентября 2021)

## Интерактивный dashboard (Tableau Public)

https://public.tableau.com/views/_17838097121720/Dashboard1?:language=en-US&publish=yes&:display_count=n&:origin=viz_share_link

Включает единый фильтр по платформе, управляющий тремя графиками одновременно:
доля запросов по часам суток, контрастные тематики, сравнение абсолюта и доли
запросов про YouTube.

## Краткие выводы

Гипотеза подтвердилась — разница между desktop и touch видна одновременно в
темах запросов, во времени использования и в характере самих запросов:

- **Desktop** используется целенаправленно — найти и, вероятно, сохранить или
  распечатать конкретный материал для работы/учёбы (учебные справочники,
  раскраски, обои для рабочего стола).
- **Touch** — канал фонового потребления контента, встроенный в бытовые
  ритуалы дня (поздравления «здесь и сейчас», утренний и вечерний скролл,
  тематические контентные рубрики).

Полное описание методологии, находок и выводов по каждому пункту задания —
в резюме внутри `аналитика.ipynb`.
